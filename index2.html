O Que é o Sigese

Desenvolvimento interno em trabalho conjunto entre SEDES e SEIMSIS para transparência ativa.

Interface pública organizada por cards temáticos (Glossário de funcionalidades).

Desenvolvimento: Kathlen Brun | Planejamento: Rodrigo Carvalho.

Acessar sigese.trt11.jus.br

O Novo Conceito do SIGESE

Framework ágil baseado em CodeIgniter, permitindo o desenvolvimento recorde de microserviços que o portal institucional não suporta.

Versionamento total via Git: rastreabilidade total, histórico de alterações e segurança, eliminando quebras causadas por plugins (ex: ChronoForms).

Independência estratégica: permite manter o Joomla limpo para atualizações oficiais, desacoplado de extensões de transparência.

Além do Coletor de Dados

Google Forms

Limitado à coleta passiva de respostas. Sem processamento de dados, sem integração profunda com bases do Tribunal e sem inteligência de regras de negócio.

SIGESE

Processamento real: CRUDs, sistemas de votação e formulários com lógica condicional avançada, integrados e rastreáveis via código.

Substituição Cirúrgica da BD

DSpace (Repositório)

Ideal para grandes volumes estáticos, mas confuso para itens que exigem contexto visual (ex: fotos de leilões) ou buscas simplificadas por certidões.

SIGESE (Metadados Estruturados)

Acesso público e sem autenticação para formatos JSON e XML.

Consultas estruturadas via parâmetros (Ano, Mês, Tipo).

Interface intuitiva focada no uso mais intuitivo.

Governança de Dados

Auditoria Nativa: Logs precisos de quem publicou, alterou ou excluiu cada arquivo ou metadado.

Hub de Integração: Capaz de conectar e exibir dados de bases externas centralizando aplicações diversificadas.

Conversão Automática: Parser que gera CSV, JSON e XML instantaneamente a partir do upload, atendendo ao Ranking CNJ.
