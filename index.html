O que é o Sigese

Desenvolvimento

  • Criado internamente em trabalho conjunto da SEDES e SEIMSIS.
  • Kathlen Brun (Dev) e Rodrigo Carvalho (Plan).

Arquitetura

  • Glossário de funcionalidades por cards.
  • Filtros dinâmicos e exportação de dados.

Acesso

  • Interface pública para o cidadão.
  • Área autenticada para publicação interna.
sigese.trt11.jus.br

Problemas Resolvidos

Joomla Legado

  • Customizações impediam atualizações mandatórias.
  • CMS em atraso crítico de segurança.

DSpace (BD)

  • Upload complexo e busca precária de arquivos.
  • Não contemplava dados transversais.

Gap Normativo

  • Ausência de APIs e Dados Abertos exigidos.
  • Licitações fora do padrão do Ranking CNJ.

Fluxo de Publicação

Input

Servidor sobe arquivo na pasta (sem formatos complexos)

Automato

Sigese faz o parser automático dos dados

Output

Disponível via API em CSV, JSON e XML

API de Transparência

Conformidade CNJ

Atende os itens 30 e 31 do Ranking de Transparência (API pública sem autenticação).

  • Suporte a CURL, Python e JavaScript.
  • Formatos legíveis por máquina.
  • Endpoints dinâmicos por ano/mês.
GET /resolucao-102/anexo-i/data
?ano=2026&mes=jan&tipo=json

// Response: 200 OK
{ "dados": [...], "versao": "1.0" }
